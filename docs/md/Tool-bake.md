# SumatraPDF bake

**Available in [pre-release 3.7](https://www.sumatrapdfreader.org/prerelease)**

Usage: `Usage: SumatraPDF bake [options] input.pdf [output.pdf]`

PDF format has annotations and widgets as special kinds of objects.

`bake` is an advanced tool that converts annotations and widgets into regular PDF objects.

## All options

```
Usage: SumatraPDF bake [options] input.pdf [output.pdf]
  -A      keep annotations
  -F      keep forms
  -O -    comma separated list of output options
```
